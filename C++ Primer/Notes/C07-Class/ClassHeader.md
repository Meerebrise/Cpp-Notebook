# 类的头文件中应该写些什么

## 头文件(.h)

1. **非模板类型**

>* 全局变量声明(带extern限定符，多次使用但要保证定义一致)

>* 全局函数的声明

>* 带inline限定符的全局函数的定义

>* 类的定义

>* 类成员函数和数据成员的声明(在类内部)

>* 类定义内的函数定义(相当于inline)

>* 带`static const`限定符的数据成员在**类内部**的初始化

>* 带inline限定符的类定义外的函数定义

## 实现文件(.cpp)

1. **非模板类型**

>* 全局变量的定义(初始化)

>* 全局函数的定义

>* 类成员函数的定义

>* 类带static限定符的数据成员的初始化
